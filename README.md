🚀 Bi2B - Landing Page de Consultoria Estratégica
---
Esta é uma Landing Page de alta performance desenvolvida para a Bi2B, focada em apresentar serviços de consultoria empresarial e converter visitantes em leads qualificados. O projeto destaca-se pela integração de análise de dados e comunicação direta.
---

🧐 Sobre o Projeto
O objetivo deste projeto foi criar uma presença digital robusta para uma consultoria, unindo design moderno com ferramentas de análise. A página foi construída com foco em UX (User Experience) e performance, garantindo que o usuário encontre as informações de serviços e entre em contato de forma rápida.

O grande diferencial é a transparência de dados, exibindo uma amostra real de dashboards para o cliente final.
---
🛠 Tecnologias Utilizadas
O projeto foi construído utilizando as melhores práticas e ferramentas do ecossistema Front-end:

React.js: Biblioteca principal para construção da interface componentizada.

Tailwind CSS: Framework de estilização utilitária para um design responsivo e veloz.

Power BI Embedded: Integração de dashboards dinâmicos para visualização de métricas comerciais.

EmailJS: Serviço de integração de formulários para envio de e-mails sem necessidade de back-end.

Vite: Ferramenta de build para um ambiente de desenvolvimento rápido.

Lucide React: Pacote de ícones leves e elegantes.
---
✨ Funcionalidades
Design Responsivo: Adaptado para qualquer tamanho de tela (Mobile, Tablet e Desktop).

Seção de Serviços: Apresentação clara de consultoria, análise de dados e registro de marcas.

Dashboard Interativo: Exibição de amostras comerciais reais via Power BI.

Formulário de Contato Inteligente: Integração completa com EmailJS, enviando os dados diretamente para a caixa de entrada do administrador.

Navegação Suave: Implementação de Smooth Scroll para melhor experiência do usuário.
